== About

Paper ‘Implementation and maintenance of automated testing in Ruby on Rails’ reviews the creation of Ruby on Rails web application framework and its most often used gems for unit testing, functional testing, integration testing and acceptance testing, test data generation and code coverage evaluation. During the course of this paper a To-Do web application was developed with the Ruby on Rails framework and automated tests were introduced using 13 different gems, including rSpec, Test::Unit, Capybara and Cucumber, in addition to comparing these gems according to several different criteria.

Keywords: Ruby, Ruby on Rails, automated testing
